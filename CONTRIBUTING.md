# Contributing

> Reporails is in beta. The fastest way to contribute is to file what you find:

- **Bugs and unexpected output** → [open an issue](https://github.com/reporails/cli/issues) with `ails version`, your OS / Python version, the command you ran, and the JSON output (`ails check -f json`).
- **Disagreement with a rule's intent** → [open an issue](https://github.com/reporails/cli/issues) describing the project shape and what you'd expect — rule definitions live in [`framework/rules/core/<slug>/`](framework/rules/core/) and we treat surface-level disagreement as signal.
- **Documentation gaps and broken links** → [open an issue](https://github.com/reporails/cli/issues) or send a PR against the relevant page in [`docs/`](docs/).
